    Each section will link to helpful articles to read for more information on the topic or site that can help with completing tasks.
    The main sections are:</p>

<ul>
    <li>Study Habits</li>
    <li>Scheduling</li>
    <li>Sleep</li>
    <li>Meditation</li>
</ul>
